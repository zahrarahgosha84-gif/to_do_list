# To-Do List CLI (Python)

A simple command-line to-do list application written in Python.  
It allows you to manage tasks with priorities, save them to a CSV file, and automatically sort by priority before saving.

## Features

- ✅ Add a new task with name, description, and priority (`high` / `medium` / `low`)
- 📋 View all tasks
- ❌ Remove a task by name
- 💾 Save tasks to `tasks.csv` (sorted by priority)
- 📂 Load tasks from a CSV file (method available, ready to extend)
- 🔁 Auto‑sort by priority: `high` → `medium` → `low`

## How to Run

1. Make sure you have Python 3 installed.
2. Save the code as `todo.py`.
3. Run it from the terminal:

```bash
python todo.py
